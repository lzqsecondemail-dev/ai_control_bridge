当前状态观察 (Current-State Observations)

以下为基于代码阅读的当前状态观察，仅为中立标签，不作为实现指令或下一版本目标。

  • 顶部状态栏可能换行：top-strip 使用 flex-wrap: wrap，在窄窗口下 Chip 可能换行，导致布局变化。
  • 中栏任务卡优先级可能不够突出：当前消息/任务卡区域与分类编辑面板大小相近，Action Steps 区域可能被大量编辑控件挤压。
  • 调试字段在普通模式下仍可见部分：部分字段（如 selected hash、project path）通过 data-acb-mode="debug" 控制，但部分字段的隐藏逻辑需确认实际行为。
  • 右栏工具卡片可能与中栏工作流竞争注意力：7 个工具卡片 + 安全锁 + Bridge/预检/就绪详情 + 测试报告占据较大纵向空间。
  • 动作步骤区域使用红色虚线框：当前使用 #ef4444 红色虚线边框，视觉上可能过于紧急。
  • Payload Detail 面板为内联面板：当前实现为 console body 底部的内联面板，不是独立的 modal overlay。
